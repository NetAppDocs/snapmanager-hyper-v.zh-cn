= 为基于SMB的Hyper-V配置SVM或CIFS服务器
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
为多个应用程序配置单个Storage Virtual Machine (SVM)或CIFS服务器可能会导致资源共享问题、进而影响Hyper-V环境。您应根据需要为基于SMB的Hyper-V配置专用SVM或CIFS服务器。
