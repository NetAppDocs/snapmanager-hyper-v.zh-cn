= 备份报告在报告名称中使用管理控制台时区信息
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


* * 问题描述 *
+
如果使用的客户端主机所在的时区与父主机不同、则在生成备份报告时、报告名称将使用客户端主机时区信息、而报告内容将使用父主机时区。

* * 发生原因 *
+
备份报告名称中的时间戳会与客户端主机时区信息一起显示。

* * 更正操作 *
+
无需采取更正操作。
