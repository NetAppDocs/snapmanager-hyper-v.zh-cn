= 添加策略
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以向数据集添加保留、计划和复制策略以及脚本、以便保护数据。

您必须具有以下信息：

* 策略名称和说明
* 保留信息
* 计划信息
* 备份选项信息
* 备份脚本信息


.步骤
. 在导航窗格中、单击菜单：Protection[Datasets ]。
. 选择要添加策略的数据集。
. 从操作窗格中、单击*添加策略*。
+
此时将显示创建策略向导。

. 完成向导中的步骤、为数据集创建保护策略。
