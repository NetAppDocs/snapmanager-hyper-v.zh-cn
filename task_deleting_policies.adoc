= 正在删除策略
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以从数据集中删除不再需要的策略。

. 在导航窗格中、单击*保护*>*数据集*。
. 选择包含要删除的策略的数据集。
. 从策略窗格中、选择要删除的特定策略。
. 在操作窗格中、单击*删除*并单击*确定*以删除此策略。
+
删除此策略还会从属于数据集的所有主机中删除已计划的作业。
