= 备份和还原通知不会在仅使用IPv6的环境中发送
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


* * 问题描述 *
+
运行纯IPv6主机时、不会收到任何备份或还原操作通知。

* * 发生原因 *
+
SMTP服务器不支持IPv6、或者它未启用IPv6。

* * 更正操作 *
+
在SMTP服务器上启用IPv6。
